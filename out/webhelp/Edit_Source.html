Edit Source

Edit Source is for users who wish to edit the raw XML or RDF code. We do not suggest that you use this option unless you are an expert.

  1. Click on the "Edit Source" toolbar icon.
  2. When you click on the edit source button, a warning pop-up appears. To proceed, click on "Yes."
  3. The edit source pop-up appears, once you have finished editing the code (again, which we do not recommend), click on the "OK" button.
