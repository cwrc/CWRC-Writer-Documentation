Introduction to tagging XML

In each document that you enter there is a pre-existing XML template that contains the basic tags necessary for that type of document. For example, in the poetry template there are "lg" and "l" tags, in which you can type or copy text into. You can play around with these pre-existing tags and/or you can add new tags.
