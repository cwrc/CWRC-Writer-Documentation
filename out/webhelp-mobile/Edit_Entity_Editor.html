Editing entities from the editor

You can add qualifying information to the RDF entity annotations added to the document. You can edit the entity annotations within the editor by right-clicking. This information can be edited at any time.

  1. Highlight the text annotatted.
  2. Right-click on the highlighted text and choose the Edit tag option.
  3. Edit any information in the pop-up you would like and click Save.
    None of the information in the pop-up is required.

The edited information will appear in the left "Entities" column. Click on the entry to see the information attached to that particular entity annotation.
